Introduction

This project is an early-stage MIDI CC to SysEx translation firmware based on the MIDIBox core and allows you to semi-realtime control the parameters of your Yamaha FB-01 4-op FM Synthesizer by sending MIDI Controller Change messages instead of SysEx, making it useable in every DAW / MIDI Sequencer.

CC Map

CC 50 to 53: OP1-4 "Volume"
CC 54 to 57: OP1-4 "Volume Fine"
CC 58 to 61: OP1-4 Response Curb
CC 62 to 65: OP1-4 Depth Level
CC 66 to 69: OP1-4 Multiplier
OP 70 to 73: OP1-4 Fine Tuning
OP 74 to 77: OP1-4 Attack Time
OP 78 to 81: OP1-4 Decay 2 Time
OP 82 to 85: OP1-4 Sustain Level
OP 86 to 89: OP1-4 Release Time

Instructions

  • Connect your PC to the USB Port of the MIDIbox core
  • Upload the .hex firmware using MIOS Studio
  • Connect FB01 MIDI In to MIDIbox core MIDI OUT2
  • Send notes to "USB" (native MIDI will be added later).
